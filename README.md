# Workspace OS

Workspace OS is a responsive productivity dashboard built with React and Vite. It includes a login-style entry screen, task and note widgets, a Pomodoro timer, weather preview, calendar view, goals, and a command palette.

## Features

- Login entry screen and workspace welcome flow
- Today’s tasks with checkbox interaction
- Quick notes panel
- Pomodoro timer controls
- Weather preview from Open-Meteo
- Weekly productivity chart
- Goal tracking cards
- Command palette with `Ctrl/Cmd + K`
- Dark mode support

## Getting Started

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Project Structure

- `src/App.jsx` — main dashboard and login UI
- `src/App.css` — application styling
- `src/index.css` — global theme and layout styles
- `public/favicon.svg` — app icon
- `index.html` — app metadata and page title
